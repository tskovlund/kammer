# RFCs — proposed designs awaiting an owner decision

ADRs in [`docs/decisions/`](../decisions/) record decisions already
made. RFCs here are the step before: a design worked out far enough to
be decided on, cheap to change because it is still a document. When an
RFC is accepted it is distilled into an ADR (and usually a milestone);
when rejected it stays here marked as such — a rejected design is
institutional memory too.
